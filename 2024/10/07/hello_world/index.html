    # C++

    • C++

      C++ 是一种广泛使用的计算机程序设计语言。它由 Bjarne Stroustrup 于 1979 年在贝尔实验室开始设计，作为 C 语言的一个扩展。C++ 提供了比 C 语言的更多功能，如面向对象编程、泛型编程和异常处理等。

    #include <iostream>

int main() {

    # Java

    • Java

      Java 是一种广泛使用的计算机编程语言，由 Sun Microsystems 公司于 1995 年发布。Java 是一种面向对象的编程语言，具有跨平台性、安全性、可移植性和高性能等特点。Java 被广泛应用于企业级应用开发、移动应用开发、游戏开发等领域。

    public class HelloWorld {

    # Python

    • Python

      Python 是一种高级编程语言，由 Guido van Rossum 于 1989 年创建。Python 是一种解释型语言，具有简洁、易读、易学等特点。Python 被广泛应用于数据分析、人工智能、网络编程、自动化脚本等领域。

    print("Hello, world!")

    # HTML

    • HTML

      HTML（超文本标记语言）是一种用于创建网页的标准标记语言。HTML 元素是构成网页的基本单位，用于描述网页的结构和内容。HTML 元素由标签和内容组成，标签用于标识元素，内容用于描述元素的内容。HTML 元素可以嵌套，形成层次结构，以描述网页的布局和结构。

    <!DOCTYPE html>
<html>
<head>
    <title>Hello, world!</title>
</head>
<body>
    <h1>Hello, world!</h1>
</body>
</html>

      JavaScript 是一种高级编程语言，由 Netscape 公司于 1995 年创建。JavaScript 是一种解释型语言，具有跨平台性、安全性、可移植性和高性能等特点。JavaScript 被广泛应用于网页开发、移动应用开发、游戏开发等领域。
